# 1. github.io 생성하기
---

자기 자신의 계정 명과 같은 이름으로

1. **계정이름.github.io**으로 repo 만들기.
2. 해당 repo를 clone하여 로컬에서 작업하기
3. 해당 로컬 내에서 html 문서를 작성하기
4. Commit & Push하기
5. https://username.github.io. 으로 접속하여 블로그가 잘 생성되었는지 확인하기

영어 메뉴얼 링크 : <https://github.io/>
